These are notable software applications for recording [[Skype]] calls. The applications may have functionality issues since the Desktop API chat functionality has been discontinued on December 2013. According to the Skype blog, call recording and compatibility with hardware devices will continue until "we determine alternative options or retire the current solution. Although chat via third party applications, will cease to work as previously communicated."
