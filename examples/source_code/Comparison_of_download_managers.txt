List of notable [[download manager]]s. A download manager is a [[computer program]] dedicated to the task of [[Uploading and downloading|downloading]] (and sometimes [[Uploading and downloading|uploading]]) possibly unrelated stand-alone files from (and sometimes to) the [[Internet]] for storage.
== Download managers ==
This comparison contains next to [[download manager]]s also [[file sharing]] applications that can be used as download managers (using the http, https and ftp-protocol). For pure file sharing applications see the [[Comparison of file sharing applications]].

''Note: Download managers in purple are no longer in development.''
