This is a list of '''file synchronization software'''. [[File synchronization]] is a process of ensuring that files in two or more locations are updated via certain rules.

== Definitions ==
'''Detect rename, move''': Programs that can not detect if a file or directory has just been renamed or moved into another folder will handle these operations like a deletion and subsequent file (re)creation, resulting in retransmitting the entire file or even directory to the mirror location at the new location or with the new name. Programs that are able to detect renames or moves (e.g. by calculating and storing  hash totals of all files contents) can avoid this by simply applying the rename or move on the mirror side, resulting in a major speedup when syncing large renamed or moved files or directories.
